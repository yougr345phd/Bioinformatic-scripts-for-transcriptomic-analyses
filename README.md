# Bioinformatic Scripts for Transcriptomic Analyses

This repository contains R scripts used for transcriptomic analysis as part of my PhD research.

## Repository contents

### Packages.R

Loads the R packages required for the analysis workflow.

### Chapter 4 scripts.R

Contains the RNA-seq analysis code for Chapter 4.

### Chapter 5 scripts.R

Contains the RNA-seq analysis code for Chapter 5

## Notes

Raw sequencing files, count matrices, metadata files, and large output files are not included in this repository.

## Software

The analyses were performed in R using packages including DESeq2, fgsea, msigdbr, org.Hs.eg.db, ggplot2, pheatmap, and tidyverse.
